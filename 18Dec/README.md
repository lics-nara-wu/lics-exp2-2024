人間が見ると読みにくいですが、文字と文字の間に以下のような記号がついています。
- `-`: 分かち書き（分割）されない箇所
- `|`: 分かり書き（分割）される箇所
- ` `: もともと空白で分かち書きされている箇所

というわけで、`.ja.tok.label` から得られる情報で、どういうときに分かち書きされ（`|`が選ばれる）、どういうときに分かち書きされない（`-`が選ばれる）のかを分類する仕組みを学習できれば、`.ja`の分かち書きされていない状態から`.ja.tok`の分かち書きされた状態に変換することができるようになるわけです。
